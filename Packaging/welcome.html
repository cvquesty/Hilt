Install Hilt

Hilt converts unlocked Windows e-Sword modules for use in e-Sword X on the Mac.

This installer places Hilt.app in your Applications folder.

  • macOS 13 Ventura or later
  • Not affiliated with e-Sword or Rick Meyers
  • Unlocked / public modules only — encrypted premium modules are refused

After install: open Hilt, add source modules, choose a destination folder, Convert, then import in e-Sword X via File → Resources → Import…
