Hilt is installed

Open Hilt from Applications (or Spotlight).

  1. Drop unlocked Windows modules (.bblx, .cmtx, .dctx, .topx) onto Sources
  2. Choose a Destination folder for converted files
  3. Click Convert
  4. In e-Sword X: File → Resources → Import…

Help is always available in the app: Help → Hilt Help (⌘?).
