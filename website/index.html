Welcome

The grip that carries the blade

A hilt is what you hold when you carry a sword. e-Sword X is the blade; Hilt is the grip that lets Mac users carry community modules with it.

Drop unlocked Windows / BibleSupport-style modules onto Hilt, choose a destination folder, convert, then import into e-Sword X. Sources are never modified.

Notarized Developer ID Direct download Unlocked modules only Offline conversion

Workflow

Four clear steps

1
Add sources Drop or open unlocked Windows modules (.bblx, .cmtx, .dctx, .topx).
2
Choose destination Pick the folder where converted files are written — never onto your sources.
3
Convert Only Ready rows convert. Encrypted or unreadable modules show a clear reason.
4
Import in e-Sword X File → Resources → Import… then restart e-Sword X if needed.

Honest limits

Hilt does not decrypt premium or product-key modules. It is not affiliated with e-Sword or Rick Meyers. Respect copyright on the content you convert.

Download for Mac

Notarized Developer ID builds you can install yourself — no App Store account required. Open the disk image and run Install Hilt, or drag Hilt into Applications.
